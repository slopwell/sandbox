# Photo Album Organizer Application

This project helps you organize your photos into albums grouped by date, with drag-and-drop reordering and tile-based previews. Metadata is stored locally in SQLite. Built with Vite, vanilla JS/HTML/CSS.

## Features
- Organize photos into albums by date
- Drag-and-drop album reordering
- Tile preview for photos in each album
- Local SQLite metadata storage (no uploads)
- Minimal dependencies, offline-capable

## Getting Started
1. `cd src && npm install && npm run dev`
2. Open the app in your browser
3. Follow the quickstart guide in `/specs/001-build-an-application/quickstart.md` (if available)

## Project Structure
- `src/` - Application source code
- `tests/` - Test code (integration, contract, unit)
- `specs/001-build-an-application/` - Feature specs, plan, tasks, research
