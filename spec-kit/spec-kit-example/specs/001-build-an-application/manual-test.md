# Manual Test: Photo Album Organizer Application

Follow the quickstart and validate all user scenarios:

1. Create a new album for a specific date
2. Add photos to the album
3. Reorder albums by drag-and-drop
4. Open an album and view photos as tiles
5. Attempt to create a duplicate album for the same date (expect error or warning)
6. Add duplicate photos to an album (expect error or warning)
7. Try to nest albums (should be prevented)
8. Test with a large number of photos in an album
9. Confirm error messages and logging are shown as expected

Check that all acceptance criteria in the spec are met.
