# Sphinx

## setup

```bash
# pyenv
curl https://pyenv.run | bash

pyenv versions
# install
pyenv install 3.13.1
# set local version
pyenv local 3.13.1

# create virtualenv
python -m venv myenv

# activate virtualenv
source myenv/bin/activate
```

## install

```bash
pip install -r ./requirements.txt
```

## sphinx-quickstart

```bash
sphinx-quickstart
```
